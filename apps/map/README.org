* Populating data

** Downloading data

- https://takeout.google.com
- deselect all
- select "Maps (your places)"
- select "Saved"
- "Next Step"

** Importing data

- move =Takeout/Maps (your places)/Saved Places.json= to =./saved_places.json=
- move =Takeout/Saved/People.csv= to =./people.csv=
- run =go run ./apps/map/cmd/mapctl -operation=import-saved-places=
- run =go run ./apps/map/cmd/mapctl -operation=annotate-peoples-places=
